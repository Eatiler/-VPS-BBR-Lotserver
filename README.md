【vps】安装BBR/Lotserver魔改加速<br>
=========================================
网络优化加速一键脚本
------------------

支持系统
Centos 6+ / Debian 7+ / Ubuntu 14+
BBR魔改版不支持Debian 8

```
wget -N --no-check-certificate "https://raw.githubusercontent.com/chiakge/Linux-NetSpeed/master/tcp.sh"
chmod +x tcp.sh
./tcp.sh
```

根据自己需求操作，重启后再使用 ` ./tcp.sh  ` 命令接着操作
